## Getting Started

## 🛠️ Setup

### 1. Clone the Repo

```bash

cd your-repo-name
git clone https://github.com/Oliver472/gymbcs.git
```

### 2. Install dependencies

```bash

npm install
```

### 3. Run the development server

```bash

npm run dev
```
